MM's 2048 Quest 🎮

Welcome to MM's 2048 Quest, a fun and addictive puzzle game where you combine numbered tiles to reach the ultimate goal: the 2048 tile! This project is a modern and stylish implementation of the classic 2048 game, built with HTML, CSS, and JavaScript.
Features ✨

    Smooth Tile Movement: Use arrow keys or swipe gestures to move tiles in four directions.

    Tile Combination: Merge tiles with the same number to create higher-value tiles.

    Score Tracking: Keep track of your score and aim to beat your high score.

    High-Score Tracker: Your high score is saved using localStorage.

    Sound Effects: Enjoy satisfying sound effects for tile movements and merges.

    Theme Switcher: Toggle between light and dark themes for a personalized experience.

    Confetti Animation: Celebrate your victory with a confetti explosion when you reach 2048!

    Mobile-Friendly: Play on both desktop and mobile devices with swipe support.

How to Play 🕹️

    Start the Game: Click the "Start Game" button to begin.

    Move Tiles: Use the arrow keys (←, →, ↑, ↓) or swipe gestures (on mobile) to move tiles.

    Combine Tiles: Merge tiles with the same number to create a new tile with their combined value.

    Reach 2048: Keep merging tiles until you reach the 2048 tile to win the game.

    Game Over: If the grid fills up and no more moves are possible, the game ends.
